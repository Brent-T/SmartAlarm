# Smartalarm

This application was made for Project 2, 3rd year Elektronica-ICT @ Odisee Gent

## Stack

* Frontend: Ionic 2 & Angular 2
* Database: MySql
* Backend: Nodejs

#### Frontend

Set up:

```
npm install
```

Start Ionic app:

```
ionic serve
```

#### Backend

Set up:

```
npm install
```

Start server:

```
node server.js
```

#### Database

The database of this project is a simple MySql-database, the scheme can be found in `./database`
